Help

Help page for the Cookbook (Flask+SQLite+SQLAlchemy) application.

Back to Home

This application runs on local server (http://127.0.0.1:5000) and uses SQLite Database server to store posted entries for a Cookbook.

It has the following features:

  1. Login (and logout): Log in to the application. Use admin/admin (or modify and apply encryption for safety!)
  2. Search: Search for entries in the database. Including Partial match search in title or text. No regex allowed.
  3. Add Entry: Add a new entry to the database.
  4. View Entries: View all entries in the database.
  5. Delete Entry: Delete an existing entry from the database. Use http://127.0.0.1:5000/delete/[entrynumber]? to delete entry list element [entrynumber] in the database. Then return Home pasting http://127.0.0.1:5000/ in the browser.
  6. Edit Entry: Edit an existing entry in the database.
